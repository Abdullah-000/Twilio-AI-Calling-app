AI Calling Demo

Start an AI-powered phone call. Provide a destination phone number, customize the assistant prompt, and pick the synthesized voice to use on the call.

Call configuration

How it works

  1. The server places an outbound call through Twilio.
  2. Twilio streams the live audio to this app over WebSockets.
  3. The bridge forwards media to OpenAI Realtime to produce conversational audio.

Expose the application using ngrok and set the public URL in your .env file so Twilio can reach the webhooks.
